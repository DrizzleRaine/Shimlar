Some things you should read before creating a character:
Login name
  • Choose a good login name. Do NOT use your char name as login name!!
  • Your login name is ONLY needed to log in the game, once logged in, your char name will be displayed.
  • Your login name MUST be unique (not used in the game by anyone else or yourself yet).

    • Password
  • Think about your password. 'Password' is not a password, and your char name isn't either.
  • Don't share your password.
  • Use a unique password for each char.
  • Don't use your password anywhere else.
  • Remember you can always retrieve a password.

    • Character name
  • Choose an appropriate name.
  • Write normally, that is start with a capital, and use only one capital.
  • Think about your name, I won't change it unless I deem it needed.
  • This name will be your name IN the game.
  • Your character name MUST be unique (not used in the game by anyone else or yourself yet).

    • Email
  • Use a valid email address.
  • You'll receive a mail with a link you'll have to click before you can enter the game (email validation).
  • If you lost your password, an email address is the ONLY way to retrieve it.

    • Security
  • You need a valid email to log in.
  • Your login name + password are needed to log in.
  • Your login name is displayed nowhere in the game (with an exception for the password recovery email).

    • Normal vs Turn based
    Shimlar actually incorporates two different games. They use the same code and database, but require a totally different strategy. You can find more info in the manual, but the main differences are here:
  • Normal: unlimited playing time, trading with others, toplist on level
  • Turn based: limited nbr turns (fights), no trading, toplist based on fame
